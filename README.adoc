==== Les fonctions
- Une fonction est définie par
  * le mot-clé *function*
  * un nom optionnel
  * une liste optionnelle de paramètres entre parenthèses
  * un corps entre accolades
- Une fonction sans nom est dite anonyme
- La propriété *name* d'une fonction donne son nom
